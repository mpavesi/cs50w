Comandos GIT


Download repositório
- git clone <url>

Incluir arquivo no git
- git add <file>

Fazer um snapshot no repositório
- git commit -am <"message">

Mostrar o estatus do reposítorio local
- git status

Fazer upload do arquivo para o repositório na nuvem
- git push

Listar as alterações
- git log

Desfazer alguma alteração
- git reset --hard <commit>
- git reset --hard origin/master


Mostar o branch atual
- git branch

Criar um branch
- git checkout -b new_branch

Alterar o branch
- git checkout master

Mostra dados do último commit
- git show

Atualiza o github
- git push origin main

Atualiza a area ded trablaho HEAD
- git pull origin main

erifica alteracoes no origin
- git remote update
